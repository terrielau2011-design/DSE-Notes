## 二、評卷參考 — 考生常犯錯誤及考評局評語（2024_MS_CP.pdf）

### A. 各題考生常犯錯誤

#### 試卷一

**第1題（選題率74.4%）— 安史之亂與唐玄宗改革**
- (a)(ii)：部分考生未能掌握「地方政制」概念，誤答府兵、十大兵鎮、三省六部、政事堂等；亦有考生誤以為十道是用作監察百姓（實為監察州刺史）
- (b)：表現未如理想者多只抄寫資料，未能按題目提供的角度（兵力部署、軍隊職責）討論
- (c)(iii)：有小部分考生混淆**李林甫**與**李輔國**，誤以為與宦官專政有關
- (d)：表現未如理想者只抄錄資料二和三，未能解說各事件因果關係，未能援引相關史事說明；又或作答玄宗荒怠政事、楊國忠與安祿山不和等與題目無關的史事

**第2題（選題率25.6%）— 洋務運動與晚清改革**
- (a)(ii)：表現較弱者只純粹抄錄資料，未加解說；又或誤以為仁壽提倡的觀點為「中學為體，西學為用」
- (c)(i)：部分考生未能寫出具體評價，只回答「評價負面」，或只抄錄漫畫文字而未能歸納作者評價
- (c)(ii)：部分考生只能泛泛指出措施令清廷人心盡失，未能具體交代史事，也未能析述史事之間的關係

**第3題（選題率30.0%）— 周代封建**
- (a)(i)：不少考生寫錯「殷」字，部分因字體太潦草致未能辨認
- (a)(ii)：少數考生將(a)(i)錯誤答案延伸寫於(a)(ii)，致損失分數
- (b)：不少考生舉證錯誤、析述含混——只籠統述說分封有助向東擴張，未引用地圖資料指出具體擴張的地理位置；忽略地圖顯示的外族位置與封國的關係；背誦封建及宗法制度內容而忽略須按地圖資料析述；引用大量封國例子（甚至遠至燕、齊、魯的東方封國）卻未扣題旨
- (c)：表現欠佳者組織混亂，表述不明，只硬背資料致冗長後仍未見提出明顯政治特色；更有甚者只抄錄表列資料含混描述；不少考生答卷夾雜砂石——將郡縣制出現、中央集權確立、世襲及世卿世祿制沒落、各國厲行變法等與題旨無直接關係的論點作延伸

**第4題 — 漢代中央制度與王莽篡漢**
- (a)(iii)：考生表現欠佳，只能抄寫資料再直接背誦戚宦輪流專權亂政的循環，未有緊扣尚書台作答
- (b)(ii)：不少考生內容疏略，或未能回應史實與東漢衰亡的關係
- (c)：不少考生反面論述能力一般，申論形式化，能展現具說服力的分析能力者寥寥可數

**第5題 — 元明中央集權**
- (a)(i)：個別答案出現錯別字，如以「洲」為「州」或以「懸」為「縣」；少數考生以左右丞相為答案，反映對元代行中書省最高長官結構有誤解
- (a)(ii)：部分考生未能準確分辨正負面影響；多以重用或貶官角度分析，忽略「行政架構」的題旨要求
- (b)：部分考生歷史基礎知識不足——明代設置廠衛的影響應以負面為主，惟不少考生縷述明代廠衛的正面影響，表現欠理想

**第6題 — 英法聯軍之役**
- (a)(iii)：少數考生誤以《南京條約》、《璦琿條約》為答案
- (b)：有考生認為入城問題是由亞羅號事件引起，或長篇大論鴉片戰爭前的貿易及外交糾紛（如阿美士德來華、林維喜事件、林則徐銷煙），對史事發生先後有所混亂
- (c)：
  - 割地賠款：部分考生將「割讓九龍半島」變成「租借九龍半島，為期99年」，或寫成「割讓九龍半島北部」或「九龍半島界限街以北」——一字之差，謬以千里
  - 內河航行權：部分考生未清楚指出內河是指哪一條河，若干誤解為黃河或珠江
  - 領事裁判權：部分考生認為領事裁判權是指在北京劃定使館區，史事混淆

**第7題（選題率11.4%）— 國共關係**
- (c)：考生忽略了當時北京學生要求停止內戰與國民政府堅持「攘外必先安內」的對立；對於西安事變後的歷史事件，尤其沒有提及各方協商後蔣介石才同意停止內戰，流於抄寫資料

**第8題 — 中華人民共和國外交**
- (a)(ii)：常見忽略美國支援國民黨對新中國政權帶來的負面影響
- (b)(ii)：不少考生混雜其他原因，如中印邊境衝突和珍寶島事件，泛泛而論，未能緊扣資料二作出回應
- (c)：表現稍遜者分析欠均衡，只偏重其中一件史事作出分析

#### 試卷二

**第1題 — 胡適與陳獨秀（單元一）**
- (a)：考生表現**欠佳**。不少考生只是抄襲題目所提供的兩項資料，略為改動數個文字，更不乏完全抄錄資料充當答案。甚少能夠條分縷析地逐項分析，能援引研習心得以作補充者更是絕無僅有
- (b)：考生表現較好（因資料三已有相當提示），惟仍有不少答案以抄錄資料為主，較少能見到考生平日研習的心得

**第2題 — 馬克思主義與毛澤東思想（單元一）**
- (a)：考生表現中規中矩。多數考生只能較詳細分析近因（十月革命以來的局面），對遠因（鴉片戰爭、自強運動、維新運動、辛亥革命未能脫困圖強等）則語焉不詳；不少考生只能抄錄資料上的句子而不知其所以然，可見考生對當時內憂外患的時代背景欠缺清晰的認識
- (b)：考生表現**欠佳**。絕大多數考生，不論是對毛澤東的矛盾論和實踐論，或是傳統文化的道家陰陽思想、王陽明的知行思想，都缺乏認識，遑論能夠將二者的共通處闡釋出來

**第3題 — 新儒家（單元一）**
- 選答此題的考生**極少**，亦欠缺較好表現者。前半問「新儒家在香港的發展」，考生在基本史實（如學者姓名、所辦院校、雜誌等）均付之闕如，更遑論新儒家學者的東西文化調和主張。後半問「改革開放後漸受重視的原因」，既欠佳卷，亦多略過不答。有考生連1978年才開始改革開放也沒有認識，卻妄言文革時種種思想鬥爭，風馬牛不相及

**第4題 — 唐代長安城（單元二）**
- (a)：表現一般者大多只簡述三方面特點，或側重個別方面，內容未夠均衡；或未有說明原理（如只述太極宮位於宮城北面，未能說明位置與皇權象徵的關係）。表現稍遜者間有概念不清或混淆史實，如將東西市功能對調、或以為唐代皇帝多在太極宮臨朝視事
- (b)：有部分考生將「關中本位政策」與「關隴集團」混為一談，概念不清；較少考生能全面闡析長安城周邊天然地形與軍事的關係；部分考生對唐代邊患認識粗疏，僅指出唐初邊患主要來自西北但未能舉出邊患名稱

**第5題 — 經濟重心南移（單元二）**
- 部分考生對「六朝」涵義理解有誤；又或誤以為北方黃河流域氣候自南朝時已變得寒冷乾燥；又或舉黃巢之亂論證北方經濟破壞；表現下者只能泛泛而論，缺乏例證，或舉一兩個史事名稱以對，未加解釋與題旨的關係——此乃學史不求甚解之弊

**第6題 — 廣州十三行（單元二，選題率2.1%）**
- (a)：考生多只複述資料內容，較少能解釋廣州繁盛的因由；對地理知識不足，鮮能解釋季候風如何有助海路貿易
- (b)：考生多未能說明十三行衰落的因由——只照抄資料所言各級官吏勒索，未有分析此現象的官場陋習背景；1856年廣州商館大火，考生大多未能指出此事件發生的背景（英法聯軍之役時英國強佔廣州城期間），更有將此事與1822年廣州十三行大火混為一談

**第7題 — 孔子（單元三）**
- (a)：考生對「仁政」理解尤佳，但對「禮治」理解一般，多未能提出令百姓知所羞恥；表現較遜者只能點出孔子政治主張的名目，或僅抄錄資料原文；不少考生解說多流於泛論，未能援引政治主張或史例說明；當中從孔子角度解釋他不出的原因，偏離題旨
- (b)：部分考生雖能理解題旨，惜未能援引相關史事說明；表現稍遜者內容夾雜不少外誤史實，如孔子辦「私人書院」以普及教育、整理「四書五經」和將弟子編緝的《論語》視為孔子整理典籍的貢獻

**第8題 — 司馬遷（單元三）**
- (a)：弘揚大義方面，不少考生未能指出《春秋》中微言大義的作用，只憑常識引伸說明，流於空疏泛論；更甚者只抄錄資料內容回應
- (b)：部分考生雖然熟悉《史記》中的例子，但忽略題旨要求就「司馬遷撰述《史記》時表現的特質」析論，只以《史記》中的例子加以概述

**第9題 — 何啟（單元三）**
- 考生整體表現**一般**。表現一般者多未能均衡回應三方面，特別欠缺史實分析何啟在改革中國方面的貢獻。部分考生在「改善香港醫療及教育」方面引例雖多，但組織混亂，或只羅列醫院和學校名稱而未能進一步析論貢獻；或列舉含有外誤的例證，如稱何啟創辦「瑪（雅）利氏醫院」、成立東華醫院、創辦聖士提反女書院、支持璞鼎查港督成立香港大學等

**第10題 — 均田制（單元四）**
- (a)：部分考生忽略題旨為「均田制施行的背景」，以不少篇幅援引均田制推行後才實施的三長制及租調制內容作答；亦有考生以北魏時的民族矛盾、漢化需要及牛疫禍害作答，甚至提及柔然威脅漸減，未能緊扣資料一立論
- (b)：史料混淆情況頗為常見——有考生以北魏均田制的畝數、隋代均田制的還授田地歲數作答，張冠李戴

**第11題 — 募兵制與徵兵制（單元四，選題率17.4%）**
- 表現一般者呈現論述失衡現象——常見忽視題幹中要求明辨募兵制和徵兵制的優點與缺點，或只側重討論宋代募兵制及明代衛所制的優劣，未能清晰地將兩者連上關係；有考生未有注意募兵制原意是挑選職業軍人故作戰力強，反而只列舉宋代招募流亡從軍而致戰鬥力低的現象；部分考生混洧明代衛所制與唐代府兵制有關授田的內容，誤把唐代均田內容寫作明代衛所屯田

**第12題 — 科舉制度（單元四，選題率14.7%）**
- (a)：表現較弱者未能把握「考試程序」的意涵，不少考生把宋代科舉的防弊措施理解為考試程序；亦有考生誤答唐代或明代的考試程序及內容
- (b)：部分考生未有理會資料提供的方向，如指出八股取士有助提高考試客觀性、進士及第後可在翰林院培訓等，卻未有回應濁流官的問題；又或指出三色卷實為明代中葉以後黨爭成因之一，凡此均不緊扣資料及題旨作答

**第13題 — 道教（單元五）**
- (a)：扣題方面有考生雖能略為引例指出道教來源的不同成分，惟在如何演變成為道教者著墨不多；又有些以「大包圍」方式作答，詳述墨家、陰陽家、儒家等與道教的關係，偏離題旨
- (b)：表現較弱者忽視題旨要求就「修撰經籍、調和儒道、調和佛道」三方面析論，只以人物方式列舉各高道對道教發展的貢獻，甚或出現張冠李戴的情況

**第14題 — 佛教（單元五）**
- (a)：小部分欠佳考卷張冠李戴——將東漢桓帝、靈帝當作唐代君主，移梁朝達摩至唐代翻譯佛經；錯別字亦不少，如把頓悟寫為「郵唔」、僧作「憎」、金「鋼」（剛）經等
- (b)：表現欠佳的考卷，在促進中國文學方面來來去去都是王維一人；為數不少的考生不懂寫「窟」，以「屈」等代替

**第15題 — 利瑪竇與天主教（單元五）**
- 考生表現**一般**。後半部分考生表現平平——部分考生雖能列舉創造主譯詞及祭孔祀祖的爭議，但所列舉史例內容貧乏、組織混亂；部分考生將明末南京教案作為清初傳教受挫例子，未能符合題旨。更甚者不少考生將**天主教與基督教混淆**，詳述清中葉基督教傳教士馬禮遜來華傳教之艱辛，當作是天主教傳教遭受挫折，完全是錯解題旨

**第16題 — 女性社會地位唐宋明（單元六）**
- 部分考生題目要求援引史實，往往只是泛泛而談，缺乏實例引證；或搬字過紙，沒有分類和解釋

**第17題 — 清末民初婦女地位（單元六）**
- (a)：表現一般的考生未能善用資料，對資料一的描述不多，未能指出《女學傳習所開學》、女學生130名等，只能略述女子接受教育數量上升；部分考生只抄寫資料二及三的內容，缺乏實例引證，作答內容失衡
- (b)：表現稍遜的考生在經濟方面的認知較弱，未能指出婦女的就業情況；少數同學更誤寫建國以來或文革時期的婦女狀況，時序大亂

**第18題 — 建國後婦女地位（單元六）**
- (a)：有考生錯把民國初年之女性為例，時序概念紊亂，反映審題未周
- (b)：表現一般——未能善用資料二及三的數據

### B. 考評局整體評論

#### 考生表現較佳的地方

1. **審題方面**：考生一般能細緻審題，理解題旨，能緊扣題目所附的資料和題義回應，並援引相關史事論證，表現出色
2. **析論方面**：考生多按題旨發揮，立場清晰，或援引相關史事支持己見，或多角度分析，或正反論證，層次分明
3. **組織方面**：考生組織答案多能按論點分段析論，輕重破立，層層遞進，條理分明
4. **時間分配**：考生能妥善安排作答時間，各題答案篇幅亦略見均衡，寥寥數行的答卷已少見

#### 考生表現仍待改善的地方

1. **審題粗疏**：仍有考生未有針對題旨作答，或抄錄題目提供的資料回應，或夾雜無關題旨的內容，雖長篇累牘，未達題目要求
2. **史事混淆**：時見考生混淆史事與人物、時序謬誤錯配、朝代概念模糊。常見例子：
   - 混淆「李林甫」與「李輔國」
   - 誤解「內河」為黃河或珠江
   - 誤將共產黨的南昌起義放在國共第一次合作期間
   - 誤以為隋代已有行省制度
   - 混淆北魏與唐代均田制的內容
   - 誤把唐代均田內容寫作明代衛所屯田
   - 混淆天主教與基督教，誤認馬禮遜為天主教傳教士
3. **字體潦草及錯別字**：部分考生字體潦草甚至無法辨認，難以評核；錯別字較多，尤其人名、地名等寫錯字，容易出現張冠李戴的錯誤：
   - 「粵漢鐵路」寫錯
   - 「張學良」寫錯「張國良」
   - 「雅麗氏」寫錯「雅利氏」或「瑪麗氏」
   - 「金剛經」寫錯「金鋼經」
   - 「列女傳」寫錯「烈女傳」
   - 「秋瑾」寫錯「秋庶」
   - 「郝建秀」寫錯

### C. 考評局建議

#### 試前準備

1. 加強基礎訓練，除深入了解史實、明白歷史發展脈絡外，亦須認清時序，避免史事混淆、人物張冠李戴及地域方位不明
2. 加強思辨、析論、比較、組織、表達等能力的訓練，建立清晰而正確的歷史觀點與概念
3. 多閱讀議論及評論性質的歷史文章，增強辨識、運用史料及論析能力；閱讀時關注人名、地名、歷史詞彙的正確書寫，避免因寫錯字而出現歧義
4. 多作論文寫作訓練，加強夾敘夾議、組織編排的訓練，從而提高分析、比較、組織史事的能力，改進表達技巧

#### 臨場應試

1. 妥善安排每題的作答時間，以免過分耗費時間於某一題目
2. 小心審題，理解題義及題目所附資料的觀點，針對題目揭示的角度，靈活運用相關歷史知識
3. 仔細思考題義，組織論點，輔以相關史實，理清脈絡，尤須注意該題應否正反立論，抑或宜作多角度討論，擬定答題綱領後逐點回應
4. 作答時清晰而有組織地表達，答案須具條理，避免偏離題旨；文字宜簡潔易明，注意正確書寫

### D. 評分準則總覽（通用框架）

所有論述題均按以下五級評分：

| 等級 | 能力表現 |
|------|----------|
| **最高級** | 答案完備，內容豐贍，能有效運用正確及相關的資料；表達清晰而具說服力，立論前後一致，顯示考生具批判及分析能力 |
| **次高級** | 內容切題及大致均衡，答案並無夾雜錯誤材料及/或重要遺漏；表達清晰，條理分明，對相關的問題能加以分析及/或提出有理據的議論 |
| **中級** | 內容基本切題及正確，但稍欠均衡；表達一般，顯示考生嘗試對試題加以議論及/或分析，但論點或前後不一致，或過分著重史實的鋪陳 |
| **次低級** | 內容夾雜重大錯誤及/或與題目不相關的資料；表達能力薄弱，論點嚴重矛盾 |
| **最低級** | 內容貧乏及/或不切題；組織混亂，表現遠離水準 |
